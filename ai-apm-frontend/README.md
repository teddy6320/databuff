# Databuff

## 环境

* 项目v2.9.1及之后的版本：  
  Node Version: v18+  
  yarn Version: v1+  

* 项目v2.9.1（不含）之前的版本：  
  Node Version: v16  
  yarn Version: v1+  

## Vite
项目在v2.10.1.1开始切换到Vite构建工具。  

包含版本：  
v2.10.1.1及之后的版本；  
v2.9.4及之后的小版本。  

安装：  
```sh
yarn
```

运行：  
```sh
yarn dev
# 或
yarn serve
```

打包：  
```sh
yarn build
```

默认地址是 `http://localhost:5173`


## Vue CLI

包含版本：  
v2.9.4（不含）之前的版本；  
v2.10.0、2.10.1.0。

安装：  
```sh
yarn
```

运行：  
```sh
yarn serve
```

打包：  
```sh
yarn build
```

默认地址是 `http://localhost:8080`
