# 更新包管理

## 页面说明

更新包管理页对应 `/config/agentPackages`，用于维护 Agent 升级包。页面展示当前可用版本列表，支持分片上传新的 ZIP 包，并支持删除旧包。

## 页面结构

| 区域 | 说明 |
|------|------|
| 列表区 | 版本号、版本特性、上传时间 |
| 上传区 | 分片上传按钮，最多保留 5 个更新包 |
| 进度弹窗 | 展示当前上传文件名和百分比 |

## 关键规则

| 规则 | 说明 |
|------|------|
| 文件类型 | 仅接受 `.zip` |
| 数量上限 | 最多保留 5 个更新包 |
| 上传方式 | 使用 `chunk-upload.vue` 分片上传 |

## 主要接口

| 接口 | 说明 |
|------|------|
| `getVersionList` | 获取更新包列表 |
| `uploadPackage` | 上传更新包 |
| `deletePackage` | 删除更新包 |

## 页面流转

1. 页面加载时先调用 `getVersionList` 获取现有版本。
2. 点击上传后进入分片上传流程，成功后刷新列表。
3. 超过 5 个更新包时，上传按钮仍显示，但会以 tooltip 明确提示需要先删除。
4. 删除操作会先进行二次确认，成功后直接从当前表格中移除。

## 注意事项

- 版本特性字段会把后端返回的换行符替换为空格，避免表格内出现多行文本撑高行高。
- 该页是部署状态模块的静态页，不在 `/config/status` 的 tab 内部切换。
