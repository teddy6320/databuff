# 日志分析模块

> 模块入口: `src/views/log/index.vue`

## 模块概述

`log` 模块当前是一个单页模块，围绕日志检索提供四类能力：

- 关键词搜索
- 主机 / 服务快捷筛选
- 日志上下文查看
- 两条日志内容对比

页面本身没有再拆独立子路由，所有操作都在 `/log` 页面内完成。

## 页面矩阵

| 类型 | 路由 | 文件 | 说明 |
|------|------|------|------|
| 页面 | `/log` | `index.vue` | 日志查询、筛选、上下文和对比入口 |

## 典型操作链路

- 关键词搜索 -> 列表刷新
- 左侧快捷筛选 -> 列表刷新并回写 URL
- 勾选两条日志 -> 打开日志对比弹窗
- 点击“查看上下文” -> 打开日志上下文弹窗
- 点击“配置说明” -> 跳转 `/config/install?type=log`

## 主要依赖 API

- `log.ts`: 日志检索与筛选条件
