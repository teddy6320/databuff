# 时间同步

## 页面说明

时间同步页对应 `/help/timeSync`，用于解释为什么集群环境需要统一系统时间，并给出外网同步、内网同步和公共 NTP 服务地址参考。

## 页面内容

| 区块 | 说明 |
|------|------|
| 背景说明 | 解释系统时间与硬件时间、集群时间一致性的意义 |
| 查看时间 | 给出 `date` 命令示例 |
| 外网同步 | 说明 `ntpdate` 与 `crontab` 的使用方式 |
| 内网同步 | 说明手动设时、硬件时间同步和内网 NTP 同步 |
| NTP 列表 | 展示国内公共 NTP 服务地址，并支持复制 |

## 依赖组件

| 组件 | 说明 |
|------|------|
| `code-view` | 展示 Shell 命令块 |
| `copy` 工具方法 | 复制 NTP 地址 |

## 页面特点

- 页面内容完全静态，不依赖后端接口。
- 表格中的 NTP 地址支持直接复制，方便运维快速使用。

## 注意事项

- 该页既是帮助中心的独立菜单页，也是部署状态页里“时间偏差过大”提示的跳转目标。
