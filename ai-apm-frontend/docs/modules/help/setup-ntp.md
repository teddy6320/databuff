# NTP 服务

## 页面说明

NTP 服务页对应 `/help/setupNTP`，用于指导用户在内网环境下搭建 NTP 服务端，并让局域网内其他机器定时与之同步时间。

## 页面内容

| 区块 | 说明 |
|------|------|
| 环境前提 | 说明服务端、客户端和网络前提 |
| 服务端配置 | 安装 NTP、修改 `/etc/ntp.conf`、启停服务 |
| 配置示例 | 给出完整 `ntp.conf` 关键片段 |
| 客户端配置 | 给出 `ntpdate` 和 `crontab` 示例 |

## 依赖组件

| 组件 | 说明 |
|------|------|
| `code-view` | 展示 NTP 配置命令与配置示例 |

## 页面特点

- 页面内容完全静态，不依赖后端接口。
- 重点面向“内网自建 NTP”场景，是时间同步页的延伸说明。
