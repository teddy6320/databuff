# 帮助中心模块

## 模块概览

帮助中心对应 `/help`，是平台内置的静态说明模块。当前已开放 3 个页面：入门指南、时间同步、NTP 服务；另外代码里还保留了 `nginxTracing` 页面实现，但路由配置已注释，当前不可达。

## 页面矩阵

| 页面 | 路由 | 说明 |
|------|------|------|
| 入门指南 | `/help/startGuide` | 引导用户快速进入主机、仪表盘、集成与用户管理 |
| 时间同步 | `/help/timeSync` | 解释时间同步背景与常用 NTP 命令 |
| NTP 服务 | `/help/setupNTP` | 说明自建 NTP 服务端与客户端配置 |
| Nginx Tracing | 未开放 | 代码存在，但当前菜单与路由未开放 |

## 模块特点

- 帮助中心页面几乎都是静态内容页，不依赖复杂表单。
- `入门指南` 是少数带业务跳转和轻量数据查询的页面。
- `时间同步`、`NTP 服务` 更接近运维知识库页面。
